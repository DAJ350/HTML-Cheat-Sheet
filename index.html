Fundamental Elements

Element Name Description
<h1>â€“<h6> Heading Used to create headings on a webpage. There a six heading sizes available in HTML.
<p> Paragraph Used to contain paragraph blocks on a webpage.
<img> Image Used to embed an image onto the webpage. This is a self-closing tag.
<video> Video Used to embed a video onto the webpage.
<a> Anchor Used to create hyperlinks on webpages. Links are created by anchoring a file path to text or another HTML element.
<embed> Embed Used to embed any media type to the webpage.
<div> Division Used to section together blocks of code.
<span> Span Used to section short lines of text inline with other text or HTML.
<ol> Ordered List Used to create a numbered list on the webpage.
<ul> Unordered List Used to create a list in no particular order.
<li> List Item Used to create list items in lists.
<strong> Strong Used to make important text render in bold font.
<em> Emphasis Used to emphasise text. Often renders text in italics.

Attributes

Attribute Name Description
src Source Used to inform the browser of where to find the media file to embed on the page.
alt Alternative Text Used to provide meaning to images. Useful for providing a description of unloaded images, increasing webpage accessibility for visially impaired users and SEO.
class Class Used to assign an element to a class. Styles can be applied to multiple elements at one time via the class selector in CSS.
href Hypertext Reference Used to specify a file path to web resourses or media files.
target Target Used to specify how the browser should open a link.
